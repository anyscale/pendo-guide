To convert your code from Workspace to a standalone job / service, either:
  • (a) Use the Convert to job / service option under the Workspace menu on the UI;
  • (b) Run: anyscale job submit -- python main.py
Anyscale will automatically snapshot the working_dir of your workspace and its settings (project, cluster environment, compute config) which can be overwritten by creating a YAML file for the job / service.
